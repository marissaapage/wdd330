Week 3 Notes

This week we talked about objects and events. we learned how to take an element from html and add an onclick event to it with javascript. Something that really interested me this week was our discussion about loops and the different ways to make them.

Ways to Make Loops

for (good)

var i;
for (i = 0; i < students.length; i++) {
    tbody.innerhtml += student[i] + "<br>";
}

.forEach (better)

students.forEach(student => {
    tbody.innerHTML +=
    `<tr>
        <td>${student.firstName}</td>
        <td>${student.lastName}</td>
        <td>${student.iNumber}</td>
    </tr>`
})

.map (best)

let studentsUpper = students.map(student =>{
    return student.lastName.toUpperCase()
})
